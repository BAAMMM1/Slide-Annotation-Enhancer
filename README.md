# OP-PDF-LectureSlides
## Description
A Tool for adding a writing area to lecture slides.

![](https://github.com/BAAMMM1/OP-PDF-LectureSlides/blob/master/teaser.jpg)

## Requirements
- JDK v21 installed on your machine

## How to run?
For Linux:
```
./gradlew run
```

For Windows:
```
.\gradlew.bat run
```
