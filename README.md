# Slide Annotation Enhancer
## Description
A Tool for adding a writing area to lecture slides pdf's.

![](https://github.com/BAAMMM1/OP-PDF-LectureSlides/blob/master/teaser.jpg)

## Requirements
- Java JDK v21 installed on your machine

## How to run?
Linux:
```
./gradlew run
```

Windows:
```
.\gradlew.bat run
```
